## Hardware- und Softwareanfordeurngen

* **64-Bit x86 Intel Core™ Solo-Prozessor oder Äquivalent, AMD Athlon™ 64 FX Dual Core-Prozessor oder Äquivalent.**
* **Internetverbindung**
* **Core-Geschwindigkeit: 1,3 GHz oder schneller.**
* **Mindestens 2 GB RAM (4 GB RAM empfohlen)**
* <a href="https://my.vmware.com/de/web/vmware/free#desktop_end_user_computing/vmware_workstation_player/15_0" target="blank">VMware oder ähnliche virtuelle Maschinensoftware (VMWare Download)</a>
* <a href="https://distrowatch.com/" target="blank">Betriebssystem (z.B. Linux Distribution)</a>

##### VMWare unterstützt folgende Betriebssysteme(32-Bit/64-Bit) :

* Windows 8
* Windows 7
* Windows Vista
* Windows XP SP3
* Windows Server 2012
* Windows Server 2008
* Windows Server 2003 Standard
* Windows XP Home Edition mit SP2
* Ubuntu 8.04 und höher
* Red Hat Enterprise Linux 4.5 und höher
* CentOS 5.0 und höher
* Oracle Linux 5.0 und höher
* openSUSE 10.2 und höher
* SUSE Linux 10 und höher

##### VMWare Alternativen:

* Virtualbox (<a href="https://www.virtualbox.org/wiki/Downloads" target="blank">Link</a>)
* Hyper-V (Bestandteil von Windows 10 Professional)
